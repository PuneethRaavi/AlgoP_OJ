## Quick Start

```bash
git clone https://github.com/PuneethRaavi/AlgoP_OJ.git
cd AlgoP_OJ
python -m venv venv
source venv/bin/activate  # For Windows: venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
